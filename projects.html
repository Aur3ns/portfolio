🔒 Déploiement d'un Active Directory sous Linux

📌 Contexte : L’entreprise NorthStar souhaitait un serveur AD sécurisé sous Linux.

🎯 Objectif : Mettre en place un **Active Directory** sous Linux avec **Samba-AD-DC** pour gérer les utilisateurs et les permissions.

🔧 Actions :

  • Installation et configuration de **Samba-AD-DC** sur Debian
  • Intégration des machines Windows et Linux dans le domaine
  • Configuration des **GPOs (Group Policy Object)** pour les restrictions et accès
  • Automatisation de la gestion des utilisateurs avec **scripts Bash et Python**

🛠️ Outils : Samba-AD-DC, Debian, Windows Client, RSAT

🛡️ Mise en place d’un SIEM avec Wazuh

📌 Contexte : Surveillance et sécurisation du réseau de NorthStar.

🎯 Objectif : Déployer un **SIEM Wazuh** pour collecter et analyser les logs afin de détecter les menaces en temps réel.

🔧 Actions :

  • Installation et configuration de **Wazuh Manager** sur un serveur Debian
  • Déploiement d’agents sur postes Windows/Linux pour la collecte des logs
  • Connexion avec **Elasticsearch et Kibana** pour la visualisation
  • Création de règles de détection d’intrusion et configuration des alertes

🛠️ Outils : Wazuh, Elasticsearch, Kibana, Linux
