🔒 Déploiement d'un Active Directory sous Linux

📌 Contexte : L’entreprise NorthStar souhaitait un serveur AD sécurisé sous Linux.

🎯 Objectif : Installer un **Active Directory** sous Linux avec **Samba-AD-DC**.

🔧 Actions :

  • Installation de **Samba-AD-DC** sur Debian
  • Configuration et intégration des machines dans le domaine
  • Mise en place des **GPOs (Group Policy Object)**
  • Automatisation de la gestion des utilisateurs avec scripts

🛠️ Outils : Samba-AD-DC, Debian, Windows Client, RSAT
